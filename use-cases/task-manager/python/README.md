## Description
A command-line task management system built in Python. It allows users to create, update, delete, and track tasks with priorities, due dates, tags, and statistics. All data is stored locally in a JSON file.

---

## Features
- Create tasks from the terminal
- List and filter tasks by status, priority, or overdue
- Update task status, priority, and due dates
- Add and remove tags
- Delete tasks
- View task statistics
- Persistent storage using JSON

---

## Technologies Used
- Python 3
- argparse (CLI handling)
- datetime (date management)
- JSON (data storage)
- UUID (task IDs)
